# Visualization
Graph is generated using sigma <http://sigmajs.org/> JavaScript libary.
- Open *ontology_visualization.html* in a web browser to automatically parse and display the ontology encoded in *ontology.json*.
- The enhanced JSON file containing all graph information can be downloaded through a link from *ontology_visualization.html*. The pre-generated graph data JSON for the project in addition is available in the root folder as *ontology_graph.json* for convenience.
